An optimal [Wordle](https://www.powerlanguage.co.uk/wordle/)-playing algorithm.

To run it, install Golang and do `go run main.go`

Example run:
```
Starting with a dictionary of 2315 words
Hold on. Computing...
Lets play...
There are 2315 possible words remaining
Your next guess should be: arise
What does the game say? 01000
There are 64 possible words remaining
Your next guess should be: north
What does the game say? 01101
There are 3 possible words remaining: humor hydro chord
Your next guess should be: cigar
What does the game say? 00002
And the answer is... humor!
```
